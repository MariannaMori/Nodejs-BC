# jest project

1. create project folder

2. create package.json
```
npm init -y
```

3. install jest as development dependency
```
npm install jest --save-dev
```

4. Modify test script in the package.json
```json
"scripts": {
    "test": "jest"
  }
  ```
5. create test folder named __tests__

6. create tests

7. run tests
```
npm test
```
